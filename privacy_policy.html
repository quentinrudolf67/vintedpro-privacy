Politique de confidentialité

Extension : VintedPro Chrome Extension

Dernière mise à jour : juin 2026

1. Résumé en deux phrases

VintedPro ne collecte aucune donnée personnelle et n'envoie rien à aucun serveur externe (sauf si vous activez la fonctionnalité IA, qui contacte l'API Anthropic avec votre propre clé). Toutes les données restent dans votre navigateur.

2. Données stockées localement

L'extension utilise chrome.storage.local pour sauvegarder uniquement :

  • Vos préférences (intervalle de bump, délai de message, etc.)
  • L'identifiant Vinted de votre compte (détecté automatiquement depuis la page)
  • La liste des acheteurs déjà contactés (pour éviter les doublons)
  • Les statistiques de ventes synchronisées depuis Vinted

Ces données ne quittent jamais votre appareil. Elles sont supprimées si vous désinstallez l'extension.

3. Données transmises à des tiers

API Vinted — L'extension effectue des requêtes vers vinted.fr en utilisant votre session de navigation existante. Aucun identifiant supplémentaire n'est transmis. Ces requêtes sont identiques à ce que vous feriez manuellement sur le site.

API Anthropic (optionnel) — Si vous activez les réponses automatiques IA et renseignez votre clé API Anthropic, les messages reçus de vos acheteurs sont transmis à l'API d'Anthropic pour générer une réponse. Cette fonctionnalité est entièrement optionnelle et désactivée par défaut. Votre clé API est stockée localement et n'est jamais partagée.

4. Données non collectées

VintedPro ne collecte pas et ne transmet pas :

  • Votre nom, email ou mot de passe Vinted
  • Votre historique de navigation
  • Vos informations bancaires ou de paiement
  • Toute donnée vers un serveur appartenant à VintedPro

5. Permissions Chrome utilisées

  • storage — sauvegarder vos préférences localement
  • alarms — planifier les bumps et vérifications automatiques
  • tabs — détecter l'onglet Vinted ouvert
  • scripting — exécuter les requêtes Vinted depuis l'onglet
  • notifications — vous informer des actions effectuées
  • host_permissions: vinted.fr — accéder à l'API Vinted
